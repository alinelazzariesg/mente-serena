⚙️ Configure o Supabase — Leia o README.md

  1. Acesse supabase.com e crie um projeto gratuito
  2. Vá em Project Settings → API
  3. Copie a Project URL e a anon public key
  4. Abra o index.html e cole nos campos SUPABASE_URL e SUPABASE_KEY
  5. Execute o SQL do README.md no SQL Editor do Supabase
